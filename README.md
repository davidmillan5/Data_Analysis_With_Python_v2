# Scientific Computing Libraries in Python

## 1. Scientifics Computing Libraries
- [Pandas](https://pandas.pydata.org/docs/) (Data Structures & Tools)
- [Numpy](https://numpy.org/doc/stable/) (Arrays & Matrices)
- [SciPy](https://scipy.org/) (Integrals, solving differential equations, optimization)

## 2. Visualization Libraries
- [Matplotlib](https://matplotlib.org/stable/users/explain/quick_start.html) (plots & graphs, most popular)
- [Seaborn](https://seaborn.pydata.org/) (plots: heat maps, time series, violin plots)
